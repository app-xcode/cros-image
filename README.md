# 🚀 Simple Proxy Server

Project ini adalah **proxy server sederhana** yang dibuat menggunakan Node.js

## 📌 Fitur
- Proxy request ke URL eksternal
- Tidak menggunakan framework (pure Node.js)
- Ringan dan mudah dimodifikasi

---

## ⚙️ Cara Menjalankan
    http://localhost:3000/?quest=https://example.com
